ID Card Maker

Press F1 or choose Help → Help to open this page.

Overview

ID Card Maker lets you preview, generate, and email membership ID cards from manual entries or a comma separated values file (CSV).

Before You Start

  • Choose a Template (Choose Template…) and a Signature (Choose Signature…) first. Editing controls, CSV import, and table selection remain disabled until both are selected.
  • A default template appears in the preview at launch. You can change it anytime.

Quick Start

  1. Choose Template… — pick your card image if you don't want to use the default (e.g. template.png).
  2. Choose Signature… — select a signature image to overlay on each card.
  3. Enter Member — full Name, ID Number (XXXXYYY), Date (YYYY-MM-DD), and Email. The preview refreshes with every change.
  4. Add member — adds the member to the bottom of the table. Click a row to edit it (button changes to Update member).
  5. Generate All Cards — saves the id card images for all rows to the output folder, which then opens.
  6. Email Selected — emails the card for the currently selected row.
  7. Email All — emails all the cards for all rows that have generated images.

Manual Entry & Editing

  • Live preview: As you type in any field (Name, ID, Date, Email), the card preview updates.
  • ID Number: Enter as XXXXYYY. The ID Number is assigned by Sentinelle when you add them. The ID Number is the current year (2025) and the next avialable number in sequece (001 to 999)
  • Date: Enter as YYYY-MM-DD. Other common formats are auto-normalized when possible. Be sure to check Sentinelle for the correct ID Number before emailing the ID Card.
  • Add vs Update:
    • Click New Member to clear fields and return to Add mode.
    • Select a table row to switch to Update mode; fields and preview populate, and clicking Update member writes changes to that row.

Comma Separated values FIle (CSV) Import

Load CSV… becomes available after selecting a Template and Signature.

The CSV must include a header row with these columns:

NameID NumberDateEmail
Jane Smith20250992025-06-30jane@example.com
  • Date handling: Several common formats are accepted and normalized to YYYY-MM-DD. Unrecognized dates are left blank.
  • After import, clicking any row populates the manual fields and updates the preview.

Save Table

  • Click Save Table to export the current rows to CSV with headers in this order: Name, ID Number, Date, Email. This can be useful for adding members to sentinelle later using copy and paste.

Generating Cards

  • Generate All Cards renders cards for every row using the currently selected Template and Signature.
  • Cards are saved to the output folder, which opens when generation completes.

Emailing Cards (Office 365)

  • Uses Office 365 email and password. On first use, sign in with Username, From email, optional From name, and Password/App Password. These credentials are saved until the program is restarted so that subsequent emailing dosn't require you to log in agian.
  • Quick sender buttons: President, Vice President, and Membership Officer can be used to autofill sender details.
  • Email Selected: Sends the card for the currently selected row. If the card image hasn’t been generated yet, it will be created automatically and attached.
  • Email All: Sends cards for all rows that already have generated images. If you haven’t generated cards, use Generate All Cards first; rows without images are skipped.

Template & Signature

  • Choose Template… selects the background image used for all cards.
  • Choose Signature… overlays a signature image on each card.
  • Both must be selected before editing entries or importing CSV.

Clear Cards

  • Clear cards requires a second click within 6 seconds to confirm.
  • It empties the output folder, clears the table and inputs, returns to Add mode, and refreshes the preview. Your template/signature selections remain.

Keyboard & Tips

  • F1 — Open this Help page.
  • Best date format: YYYY-MM-DD.
  • Status bar: Watch the status bar for confirmations, progress, and any errors.

Troubleshooting

  • Editing controls are disabled: Choose both a Template and a Signature.
  • Preview is blank: Choose a Template.
  • Email All skipped some rows: Generate cards first (or use Email Selected to send an individual card, which will generate the image if needed).
  • Date didn’t print: Enter it as YYYY-MM-DD.

Privacy & Storage

  • SMTP credentials are used to send email and are not written to disk by the app.
  • Generated cards and exported CSVs are saved in your local output folder.
