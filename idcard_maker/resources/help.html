Overview

ID Card Maker creates membership ID cards from member information.

  • Preview shows what the card will look like for the currently selected row.
  • Members table is where you enter or load member information.
  • Create card(s) saves ID cards for selected rows.
  • Email card(s) sends emails (with cards attached) for selected rows.

Recommended workflow

  1. Choose the ID card image (required) and Signature image (required). If you want, check Set as default so they load automatically next time.
  2. Choose the Save folder (required). If you want, check Set as default so it loads automatically next time.
  3. Add members by loading a CSV or clicking Add member and typing directly into the table.
  4. Select the rows you want to work on using the checkboxes (or Select all).
  5. Click Create card(s) to save ID cards.
  6. Click Email card(s) to email cards to members.
  7. Use Open folder to view cards, and Export CSV to save your updated list.

Buttons turn on as you go: Some buttons are disabled until you have chosen the required files/folder and filled in the required member details.

Settings (ID card & Signature)

The ID card image is the background for the card and is required. The Signature image is placed on top of the card and is required.

  • Choose file... selects an image from your computer.
  • Set as default saves your selection so it loads automatically next time.
  • If you change the ID card image or signature later, the preview and new cards use the latest selections.

Members table

Member information is entered and edited directly in the table.

Field What it’s used for Notes
Name Printed on the card and used in email templates. Use the full name.
ID Number Printed on the card; used as the base filename; required for create/email. Format: 7 digits (example: 2010031).
Date Printed on the card. Required format: YYYY-MM-DD (example: 2026-04-20).
Email Destination for emailing cards. Required; must look like name@example.com.

All four fields are required for any row you want to create or email.

  • Add member inserts a new empty row.
  • Remove row deletes a row using the Remove button at the far right of that row.
  • Preview a row by clicking it. (This is separate from the checkboxes.)
  • Select work rows using the checkbox column; only checked rows are created/emailed.

Buttons (what to click and when)

The buttons above the members table are ordered to match the recommended workflow:

  1. Add member — inserts a new empty row.
  2. Create card(s) — creates ID cards for the checked rows.
  3. Email card(s) — sends emails for the checked rows.
  4. Open folder — opens the current save folder.
  5. Export CSV — saves a CSV file containing the current table rows.
  6. Clear cards — deletes saved ID cards from the save folder. (You must click it twice to confirm.)

Load and export CSV

Load (Load members CSV)

A CSV is a simple spreadsheet file (you can open it in Excel). Your CSV must have a first row with these column names:

nameid_numberdateemail
Jane Smith20100312026-04-20jane@example.com
  • Dates are normalized to YYYY-MM-DD on import when possible. You can only create/email when the Date column contains valid YYYY-MM-DD values.
  • After import, use the checkboxes to choose which rows to create/email.

Export (Export CSV)

  • Exports the current table as CSV so you can save it and load it again later.
  • Tip: Export after you have added rows or corrected emails/dates.

Creating card(s)

  • Choose the ID card image, Signature image, and Save folder first.
  • Check the rows you want to create, then click Create card(s).
  • ID cards are saved as files (one per selected row), using ID Number as the base filename.
  • If something is missing or not valid in a checked row, it will have a red border.

Emailing card(s)

Email sending uses the configured active email account (Microsoft or Gmail).

  1. Open Email settings at the top of the app.
  2. Choose the active account: Microsoft or Gmail.
  3. Choose the sender (President, Vice President, or Membership Officer). The message will switch to match that sender.
  4. Optional: turn on Union Management System and enter its email address to send a copy of every email.
  5. Enter Email and Password for the active account. (An app password may be required.)
  6. Optionally set From name, and customize the Subject and Message.
  7. Click Save. Then click Email card(s) above the members table.
  • Only checked rows are emailed.
  • Every checked row must have all required fields filled in.
  • Create the ID cards first. Emailing will only attach cards that already exist in the Save folder.

Fill-in words for your email

You can use these fill-in words in your email Subject and Message:

  • {name}, {id_number}, {date}, {email}, {sender}

Default SMTP server settings

  • Microsoft: smtp.office365.com:587 (STARTTLS)
  • Gmail: smtp.gmail.com:587 (STARTTLS)
  • If your organization uses different SMTP settings, follow the instructions from your administrator.

Save folder (required)

The save folder is where ID cards are saved. When you email cards, the app looks in the save folder to find the ID cards to attach.

  • Use Choose folder... to pick a folder.
  • Set as default saves the save folder so it loads automatically next time.
  • Open folder opens the save folder so you can see your ID cards.

Keyboard & tips

  • F1 - Open this Help page.
  • Preview vs selection: Clicking a row selects it for preview; checkboxes control what gets created/emailed.
  • Status messages: Watch the “Batch” and “Email” status lines for summaries and errors.

Troubleshooting

  • Create/Email buttons are disabled: Choose the ID card image, Signature image, and Save folder. Then check at least one row and fill in all required fields.
  • Email says to enter email + password: Open Email settings, choose the active account, enter your email and password, click Save, then try again.
  • Email login failed: Double-check your email and password. If you use two-step sign-in, you may need an app password. Some workplaces also block email sending unless IT turns it on.
  • Date did not print: Use YYYY-MM-DD.
  • Preview is blank: Choose the ID card image and Signature image, then click a row to preview it.

Privacy & storage

  • Your email address and password are used to send email. For safety, the app does not save your password.
  • Created ID cards and exported CSV files are saved on your computer.
